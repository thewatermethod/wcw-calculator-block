This is a WYSIWYG build your own cost Calculator block. (Great for allowing customers to calculate things like tuition and program cost)

== Description ==

This plugin allows you to configure a user facing costs Calculator. If you have a summer camp, study abroad program, product, or anything else where you might want to 
demonstrate the costs to your customers, this plugin allows you to do that, all highly editable and configurable through the block editor interface.
